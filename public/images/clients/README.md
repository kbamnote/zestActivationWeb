# Client Logo Images Directory

This directory should contain all client company logo images.

## Image Requirements

- **Format**: PNG (preferred for transparency) or SVG
- **Size**: Maximum 200x80 pixels (logos will be scaled to fit)
- **Naming Convention**: Use lowercase with underscores (e.g., `tata_motors.png`, `hdfc_bank.png`)
- **Background**: Transparent background recommended
- **Quality**: High-resolution for crisp display on all screens
